1. Известно что для архивации файлов существует пакет gzip, который работает только с файлами. 
Также есть пакет tar, который может объединять файлы в один. 
Приведите команду, которая запаковывает все файлы с расширением log в каталоге /home/de12/de/files в архив quiz.tar.gz, 
при этом использует максимальную степень сжатия.

cd /home/de12/de/files/|tar czf /home/de12/alpa/quiz.tar.gz *.log


2. Имеется скрипт quiz.sh:
[de1m@data-engineering ~]$ ls -l /home/myuser/
-rw-r--r-- 1 myuser mygroup 1024 Jan 26 14:21 quiz.sh

a) файл могли запускать все
   chmod ugo+x quiz.sh 

b) файл могли изменять владелец и группа
   chmod g+w quiz.sh

c) все пользователи сервера, не являющиеся владельцем и не входящие в его группу, не могли читать этот файл
   chmod o-r quiz.sh


3. Напишите bash-скрипт, сохраняющий все запущенные процессы пользователя de12 в файл processes.txt. 
В задаче есть допущение что имя процесса не может содержать слово de12.
#!/bin/bash

ps -u de12 | cut -c 1-8,26- >> /home/de12/alpa/processes.txt
cat /home/de12/alpa/processes.txt



4. Напишите bash-скрипт, сохраняющий подробное описание объектов, расположенных в каталоге /home/de12, 
но не принадлежащих пользователю de12 (владелец не de12) в файл description.txt. 
В задаче есть допущение что имя файла или группы не может содержать слово de12.
#!/bin/bash

ls -la /home/de12/|grep -i -v de12 >> description.txt
